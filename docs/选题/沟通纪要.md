## 1. 已确认事项

1. **赛道与方向**
   - 采用：瑞芯微赛道

2. **项目定位**

3. **预算与展示要求**

   - 主环境：**Ubuntu（双系统）**
   - Windows：辅助使用（文档、备用工具）
   - 官方资料中未发现必须仅在 Windows 执行的步骤

## 2. 作品名称与简介（当前版本）

### 2.1 作品名称

- 基于瑞芯微RV1126B的工业数据采集与AI告警网关

### 2.2 作品简介（提交版）

本作品面向工业设备状态监测与预防性维护，基于瑞芯微Linux平台设计一体化边缘智能网关。系统接入振动、温湿度、电流等多源传感器，以毫秒级周期采集关键运行参数，并在本地HMI界面实时展示设备状态、趋势曲线和告警事件。算法采用“阈值规则+AI异常检测”双通道机制：规则链路保障低时延与稳定性，AI链路识别早期异常并输出风险分数，实现黄警/红警分级预警。终端支持声光联动告警、日志存储与历史回溯，可用于设备巡检、现场值守和维护决策。作品突出软硬结合、工程化落地与可扩展能力。
