# 2026-Embedded-Competition（团队协作仓库）

本仓库用于嵌入式竞赛项目的方案管理、采购管理、计划排期与后续代码开发协作。

## 目录结构

```text
.
├── docs/
│   ├── 瑞芯微主线/          # 主赛道方案、计划、分工、瑞芯微赛题
│   │   └── 赛题/
│   ├── 采购/                # 采购与进度看板
│   ├── 报名材料/            # 报名相关文档
│   └── .archive/其他赛道/   # 其他赛道官方资料归档（低可见度）
└── scripts/                 # 临时脚本与自动化脚本
```

## 关键文档入口

- 瑞芯微项目总方案：`docs/瑞芯微主线/项目方案_瑞芯微_AI工业数据采集与本地HMI网关.md`
- 采购看板：`docs/采购/边缘网关项目采购与进度看板.xlsx`
- 采购负责人：`docs/瑞芯微主线/采购负责人分工与到货时间.md`
- 半周计划：`docs/瑞芯微主线/每半周执行计划_12周.md`
- 分工与验收节点：`docs/瑞芯微主线/三人分工与关键验收节点.md`

1. 新增文档默认放 `docs/` 对应子目录。
2. 临时脚本放 `scripts/`，稳定后再迁移到正式目录。
